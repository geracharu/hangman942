# Hangman
Hangman is a classic game in which a player thinks of a word and the other player tries to guess that word within a certain amount of attempts.

This is an implementation of the Hangman game, where the computer thinks of a word and the user tries to guess it.

## Introduction

This project aims to enhance fundamental Python programming skills by completing a series of tasks related to lists, user input, and random number generation.

## Project Objectives

- Gain proficiency in creating and manipulating lists
- Understand the concept of user input and how to handle it in Python
- Utilize the random module to generate random numbers and perform random selections

## Project Tasks

1. Create a list of your 5 favorite fruits
2. Randomly select a fruit from the list and display it
3. Prompt the user to enter a single letter and validate their input
4. Implement basic error handling to ensure valid user input
